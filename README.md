
# 🛒 Análisis de Embudo y Retención: MercadoLibre
Data Analysis Project | E-commerce & Product Analytics

## 🎯 ¿Por qué lo hice? (El Problema)
En el competitivo mercado del E-commerce, entender dónde se pierden los clientes es vital. El objetivo de este proyecto fue auditar el flujo de comportamiento de los usuarios en MercadoLibre (01/2025 - 08/2025) para identificar fricciones críticas en el proceso de compra.

El análisis busca responder: ¿Por qué el 85.7% de los usuarios que seleccionan un producto no lo agregan al carrito? Identificar y resolver estos cuellos de botella permite optimizar la inversión publicitaria y aumentar las ventas sin incrementar el tráfico.

## 🛠️ ¿Qué hice? (Acciones)
Implementé un análisis integral de producto utilizando datos de comportamiento de usuario:

* Análisis de Embudo (Funnel Analysis): Rastreé el viaje del cliente desde la first_visit hasta la purchase para localizar la mayor caída porcentual.

* Segmentación Geográfica: Comparé la eficiencia de conversión entre países (Uruguay, México, Brasil, etc.) para detectar mercados con bajo rendimiento.

* Análisis de Cohortes y Retención: Evalué la lealtad del usuario a 7, 14, 21 y 28 días para entender cuándo y por qué dejan de usar la plataforma.

* Propuesta Estratégica: Diseñé recomendaciones basadas en datos para mitigar la fuga de usuarios en etapas críticas.

## 🚀 Resultados e Impacto (Métricas Clave)
A través de este análisis, se obtuvieron hallazgos con impacto directo en la estrategia comercial:

* Identificación del Punto de Quiebre: Se descubrió que el salto de "Selección de Producto" a "Agregar al Carrito" es el punto de mayor fuga, con una pérdida masiva del 85.7% de los usuarios.

* Eficiencia de Conversión Final: La tasa de conversión total (Compra) se situó en un 1.25%. Sin embargo, mercados como Uruguay (4.5%) y México (2.4%) superan el promedio, revelando mejores prácticas aplicables a otros países.

* Diagnóstico de Retención: Se detectó una caída crítica en la retención hacia el Día 28 (cayendo a un 1.6% - 3%). Esto indica una falta de incentivos para el hábito de compra recurrente después de la tercera semana.

* Detección de Crisis en Brasil: Se identificó a Brasil como el mercado con mayor área de oportunidad, presentando la conversión final más baja (0.68%), lo que sugiere la necesidad de revisar costos de envío o métodos de pago locales.

## 🧰 Herramientas Utilizadas
* Análisis de Datos: Excel / CSV (Procesamiento de Resumen Ejecutivo).

* Métricas analizadas: Tasas de conversión, Análisis de Cohortes, Retención D7-D28.

* Enfoque: Estrategia de Crecimiento (Growth) y Optimización de Conversión (CRO).
